app-kernel
==========

Название
--------

    app-kernel - пакет содержит компоненты для создания модульного приложения

Описание
--------

Пакет `app-kernel` содержит содержит компоненты для создания модульного приложения.

Компоненты
----------

- [app-kernel/App](doc/app-kernel_App.md)

Модули
------

- **MOD_ENV**   - модуль для загрузки переменных среды из файла `.env` и окружения
- **MOD_LOG**   - модуль для вывода логов приложения
- **MOD_CMD**   - модуль управления командной строкой
- **MOD_HELP**  - модуль для вывода справочной информации
- **MOD_CONFIG** - модуль вывода настроек приложения
- **MOD_LD**    - модуль для загрузки динамических библиотек (\*.so)
- **MOD_ASYNC** - модуль инициализирует асинхронную среду tokio (требует std)

Требования
----------

- поддерживает компиляцию с `std` и `no_std`
