### Описание

*app* - базовое приложение. Предназначено для загрузки функциональных модулей из
динамических библиотек .so. Сам по себе *app* умеет только:
- запускать асинхронную среду tokio
- создавать соединение с бд PostgreSQL
- загружать динамические библиотеки .so
- отправлять события в модули
- загружать настройки из конфигурационного файла, командной строки и переменных
  среды
- выводить отладочную информацию по работе модулей

*app* содержит модули:
1. __MOD_ASYNC__ - запуск асинхронной среды tokio
2. __MOD_DB__ - соединение с бд PostgreSQL
3. __MOD_LD__ - загрузка динамических библиотек *.so
4. __MOD_ENV,MOD_LOG,MOD_CMD,MOD_HELP,MOD_CONFIG__ - вспомогательные модули

### Сборка

`make build-bin-app` - создание приложения app  
`make build-lib-app-example-a` - пример асинхронного модуля с работой бд  
`make build-lib-app-example-b no_std=1` - пример модуля с *no_std*  

### Команды

`target/release/app` - вывод справки программы *app*  
`target/release/app -L app_a a` - загрузка библиотеки libapp_a.so и выполнение
команды *a* из модуля *MOD_A* (модуль *MOD_A* подзагрузит библиотеку *MOD_B*
автоматически так как прописана зависимость модуля *MOD_A* от *MOD_B*)  
`target/release/app -L app_b b` - загрузка библиотеки libapp_b.so и выполнение
команды *b* из модуля *ModuleB*
